# Optimum SCS React + Vite Project

## Stack
- React
- Vite
- React Router
- Plain CSS

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Routes
- /
- /solutions
- /industries
- /about
- /case-studies
- /contact
- /fee-quote

## Notes
- Uses one consistent container width
- Shared navbar and footer across all pages
- Pure CSS glassmorphism styling
- Decorative non-text background only
